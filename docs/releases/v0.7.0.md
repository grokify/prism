# v0.7.0 Release Notes

**Release Date:** 2026-05-18

## Overview

This release marks a major restructuring of the PRISM repository. The maturity modeling functionality previously in this repository has been moved to [prism-intelligence](https://github.com/grokify/prism-intelligence), and this repository now serves as the **unified orchestration layer** for the PRISM ecosystem.

## Breaking Changes

!!! warning "Repository Repurposed"
    The previous `github.com/grokify/prism` functionality (maturity modeling, SLIs, SLOs) is now in `github.com/grokify/prism-intelligence`. Update your imports accordingly.

**Migration:**

```go
// Before (v0.6.0 and earlier)
import "github.com/grokify/prism"

// After (v0.7.0+)
import "github.com/grokify/prism-intelligence"
```

## What's New

### Ecosystem Orchestrator

The prism repository now provides a unified orchestration layer that integrates:

- **prism-capability** - Capability stacks and dependencies
- **prism-intelligence** - SLIs, SLOs, and maturity models
- **prism-execution** - OKRs, roadmaps, and initiatives

### Cross-Module Queries

```go
import "github.com/grokify/prism/ecosystem"

eco := ecosystem.New(config)

// Get full context for a capability
ctx := eco.GetCapabilityContext("sast")

// Query all metrics across documents
metrics := eco.AllMetrics()

// Query all initiatives across roadmaps
initiatives := eco.AllInitiatives()
```

### Ecosystem Types

- `Config` - Configuration for loading ecosystem files
- `Ecosystem` - Aggregates all PRISM module documents
- `Statistics` - Summary statistics across modules

### CLI

The `prism` CLI provides command-line access to ecosystem operations:

```bash
# Load an ecosystem and view statistics
prism ecosystem load --dir ./ecosystem

# Validate all documents
prism validate --config prism.yaml

# Show ecosystem statistics
prism stats --dir ./ecosystem --json
```

Available commands:

- `prism ecosystem load` - Load an ecosystem from config or directory
- `prism validate` - Validate all documents and cross-references
- `prism stats` - Show ecosystem statistics
