## [v0.7.0] - 2026-05-18

### Highlights

- Repository repurposed as PRISM ecosystem orchestrator
- prism-intelligence (maturity modeling) moved to separate repository
- Unified orchestration layer connecting prism-capability, prism-intelligence, and prism-execution
- CLI for ecosystem loading, validation, and statistics

### Breaking

- Previous prism functionality (maturity modeling) moved to github.com/grokify/prism-intelligence ([`d93378f`](https://github.com/grokify/prism/commit/d93378f))
- This repository now serves as the ecosystem orchestrator, not the maturity library ([`d93378f`](https://github.com/grokify/prism/commit/d93378f))

### Added

- Ecosystem package with unified orchestration types ([`d93378f`](https://github.com/grokify/prism/commit/d93378f))
- Config type for loading ecosystem configuration ([`d93378f`](https://github.com/grokify/prism/commit/d93378f))
- Ecosystem type aggregating CapabilityStacks, PRISMDocuments, OKRSets, and Roadmaps ([`f44e15b`](https://github.com/grokify/prism/commit/f44e15b))
- Cross-module queries: GetCapabilityContext, AllMetrics, AllInitiatives ([`f44e15b`](https://github.com/grokify/prism/commit/f44e15b))
- Cross-module validation for capability → SLI references ([`f44e15b`](https://github.com/grokify/prism/commit/f44e15b))
